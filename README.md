# Prédiction des prix immobiliers en Californie

## Objectif
Prédire le prix médian des maisons californiennes
à partir de données socio-économiques avec sklearn.

## Démarche
- Nettoyage des données (valeurs NaN, valeurs aberrantes)
- Création de nouvelles features (ratio chambres, zone côtière...)
- Comparaison de 6 algorithmes de régression
- Optimisation des hyperparamètres avec GridSearchCV

## Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Seaborn
